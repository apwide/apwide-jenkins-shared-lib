<p>Update Apwide Golive Environment status. Reuse JIRA Global Environment Variable and for full protocol, consult jira global variable</p>
<p></p>
<dl>
    <dt>application</dt><dd>Environment Application name. (if null, read from env.APW_APPLICATION)</dd>
    <dt>category</dt><dd>Environment Category name. (if null, read from env.APW_CATEGORY)</dd>
    <dt>version</dt><dd>Deployed version name</dd>
</dl>
<p>Usage example:</p>
<pre><code>
apwStatusChanged application:'eCommerce', category:'Dev', status:'Up'
</code></pre>

<p>Usage if environment variables are defined:</p>
<pre><code>
apwStatusChanged status:'Up'
</code></pre>
